# Password Manager in Python

A simple command-line based Password Manager built using Python.  
It securely stores account credentials in a JSON file using basic encryption and protects access with a master password.

---

## Features

- Master password protection  
- Add new account credentials  
- View saved passwords  
- Update existing passwords  
- Delete accounts  
- Stores data safely in a JSON file  
- Uses basic encryption & decryption logic
